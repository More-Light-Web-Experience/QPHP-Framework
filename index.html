QPHP-Framework

a simple and light php framework for new phpers. There are so many php framework now, some large some simple, we sometimes have no ideas how to select a good one for us. Especially for fresh phper like me. Now I create this simple and light php framework for all phpers who have never used any php framework but want to learn some basic skills for framework.
